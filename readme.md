# Toxicity-Text APP

O app utiliza a biblioteca TensorFlow para analisar textos de usuários e retornar uma resposta se eles são ofensivos ou não.

## Funcionalidades

- **O Que o App faz?**
  - Análise de Sentimento;
  - Detecção de Ofensas;
  - Classificação Final;
  - Interface Simples.
